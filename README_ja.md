# セキュアマルチVLANネットワークラボ

*小規模企業ネットワークラボ*



このラボは、Cisco Packet Tracer を使用して基礎的なエンタープライズネットワーキングとセキュリティの演習を行うために作成されました。



---



## 主な機能



* **マルチVLAN設計:** 複数の VLAN による論理的なネットワーク分割

* **レイヤ3バックボーン:** コアスイッチとルーター間で OSPF ルーティングを設定

* **セキュリティとポリシー:**  

    * **ACL適用:** VLAN間の隔離と最小権限アクセスを実現  

    * **アンチリコネサンス:** 内部ルーターインターフェイスおよび DMZ サブネットへの ICMP をブロック  

    * **管理用制御:** 専用 VLAN にアクセス制御ポリシーを設定  

    * **ポートセキュリティ:** 全アクセスポートに有効化

* **インターネットアクセス:** NAT（オーバーロード）を設定  

* **DMZ:** サーバー用に分離されたゾーン



---



## 設計意図と検証目的

本検証ラボは、3階建ての中小企業を想定したネットワーク設計であり、以下の設計目標を含んでいます。

* **セキュリティ設計：** 各フロアにVLANを分離し、外部接続を禁止した管理VLANを配置。
* **アクセス制御：** Extended ACLにより、セグメント間のアクセス制御とDMZへの接続を実装。
* **冗長性と接続性：** OSPFルーティングを採用し、フロア間通信の安定性と耐障害性を確保。
* **コスト効率：** 1階にはルーターとL2スイッチ、2・3階にはL3スイッチを配置。
* **運用管理：** Port Security、PortFast、BPDU Guard を設定し、セキュリティと運用効率を向上。
* **リモートアクセス：** 全デバイスをSSH経由でのみ管理可能に設定。
* **NAT処理：** 外部ネットワークへの接続VLANにはNAT Overloadを適用。

## 使用技術



VLAN、SVI、VLAN間ルーティング、OSPFv2、ROAS、DHCP、SSH、NTP、NAT（オーバーロード）、標準/拡張 ACL、ポートセキュリティ

## デバイスアクセス / パスワード

本ラボの全デバイスはパスワードで保護されています。SSHやログイン時は以下の情報を使用してください：

- **ユーザー名:** `admin`  
- **パスワード / シークレット:** `1234`  
- **特権EXEC（enable）:** `abcd`  

Cisco Packet Tracer 上でデバイスにアクセスする際に使用してください。

## 注意事項



* **設定ソース:** `/configs` フォルダー内のファイルは仮想デバイスの `show running-config` 出力をそのまま使用しています  

* **ファイル使用:** `.pkt` ファイルは **Cisco Packet Tracer** で開く必要があります  

* **ACL命名:** ACL は機能に応じて分かりやすく命名されています（例: SSH ACL は `SSH`）  


* **目的:** このラボは学習および演習目的です
